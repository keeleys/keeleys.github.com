ArrayList

一个泛型,实现了序列化接口,拷贝接口,随机访问接口

RandomAccess

只是一个标记接口,这个接口源码有注释,大体意思是实现类可以通过类似数组下标方式的快速访问,这样比只能通过顺序访问的效率高.

add

添加的时候先分配空间,然后直接数据最后的下标赋值,空间不足的时候还会copy

指定的下标分配空间的时候, 先判断下标的范围.然后分配空间
这里会调用native方法复制数组,把index后面的所有元素往后复制,空出需要执行的index

remove,直接把index+1之后的元素往前复制一位,然后丢弃最后一个元素.
